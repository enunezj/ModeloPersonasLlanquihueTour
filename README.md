Descripción general del sistema

Proyecto desarrollado en Java para modelar personas vinculadas a Llanquihue Tour, aplicando Programación Orientada a Objetos mediante: Encapsulamiento, composición, herencia, utilizacion de constructores, getters y setters, con sobreescritura del metodo toString.

Estructura general del proyecto

```plaintext
📁 src/
├── app/         # Clase principal con el método main
└── model/       # Clases de dominio (Persona, Direccion, Empleado.)
````

Instrucciones para clonar y ejecutar el proyecto

1. Clonar el repositorio desde GitHub:

```bash
git clone https://github.com/enunezj/ModeloPersonasLlanquihueTour
```

2. Abrir el proyecto en IntelliJ IDEA.

3. Ejecuta el archivo `Main.java` desde el paquete `app`.
